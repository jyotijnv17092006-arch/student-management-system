#  Student Management System (C++)

##  Overview
A simple console-based Student Management System built using C++.
It supports CRUD operations and stores data using file handling.

##  Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student

##  Tech Used
- C++
- STL (vector)
- File Handling

##  How to Run
g++ main.cpp  
./a.out

##  Files
- main.cpp
- students.txt
